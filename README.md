## Installation

1. Download the [Arduino IDE](https://www.arduino.cc/en/software) software
3. Clone the Repo
   ```sh
   git clone https://github.com/DarioArzaba/ArduinoScripts.git
   cd ArduinoScripts
   ```
4. Open, edit and run any of the `ino` files provided.
